# Example Implementations

This directory contains reference implementations and demo projects that showcase practical applications of Night Market plugin patterns.

## Available Examples

### Context Optimization Demo
[context-optimization-demo.md](./context-optimization-demo.md)

A full demonstration of MECW principles and context optimization techniques including:
- Progressive loading techniques
- Subagent delegation patterns
- Context pressure monitoring
- Performance benchmarking

**Related Plugin**: [Conservation](../../plugins/conserve/)

### Async Web Scraper Demo
[async-web-scraper-demo.md](./async-web-scraper-demo.md)

A complete async/await Python example for concurrent web scraping showing:
- Concurrent HTTP requests with aiohttp
- Rate limiting and connection pooling
- Error handling and retries
- Data extraction patterns

**Related Plugin**: [Parseltongue](../../plugins/parseltongue/)

## Using These Examples

These examples serve as:

1. **Learning Resources** - Understand how patterns are applied in practice
2. **Implementation References** - Copy and adapt code for your projects
3. **Testing Templates** - Use as starting points for your own demos
4. **Integration Guides** - See how skills combine in real workflows
